# 🧠 General Knowledge Quiz (Python)

## Description
This is a simple command-line General Knowledge Quiz application developed in Python as part of the DecodeLabs Python Internship Project 4. The application presents multiple-choice questions to the user, checks the answers, calculates the score, and displays the final result with performance feedback. It demonstrates the use of loops, conditional statements, lists, dictionaries, and user input handling in Python.

## Features
- Multiple-choice quiz questions
- Instant answer validation
- Automatic score calculation
- Displays final score and percentage
- Performance feedback based on the score
- Simple menu-free console interface

## Technologies Used
- Python 3

## How to Run
1. Download or clone the repository.
2. Open the project folder.
3. Run the Python file:
   ```bash
   python project4_quiz.py
   ```

## Learning Outcomes
- Working with lists and dictionaries
- Using loops and conditional statements
- Handling user input
- Score calculation and result generation
- Building an interactive console-based application
